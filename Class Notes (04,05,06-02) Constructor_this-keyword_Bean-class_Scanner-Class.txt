Constructor:- 
===========
It is a special type of non-static method that is executed automatically only when an object is created.
					OR

In Java, a constructor is a special block of code that is called when an object of a class is created. It initializes the object and sets its default values.



Note:- If you don’t write a constructor, Java automatically creates a default one.
-------
Types of Constructor:-
=====================
- Default constructor:-
-----------------------
	A default constructor is a constructor created by the compiler if we do not define any constructor(s) for a class. 	

- No-argument constructor:-
--------------------------
public class Student {
	String name;
	int rollNumber;
	String collegeName;
	
	
	public Student() {
		System.out.println("Inside no-argument constructor......");
	}
}


- Parameterized Constructor:-
---------------------------
public class Student {
	String name;
	int rollNumber;
	String collegeName;

	public Student(String name, int rollNumber, String collegeName) {
		this.name = name;
		this.rollNumber = rollNumber;
		this.collegeName = collegeName;
		System.out.println("Inside Parameterized Constructor.....");
	}
}

Constructor overloading:-
=======================
package com.chitkara;

public class Student {
	String name;
	int rollNumber;
	String collegeName;
	
	public Student() {
		System.out.println("Inside no-argument constructor....");
	}
	public Student(String name) {
		this.name = name;
		System.out.println("Inside Constructor where only passing one argument `name`.....");
	}
	public Student(String name, int rollNumber) {
		this.name = name;
		this.rollNumber = rollNumber;
		System.out.println("Inside Constructor where passing two argument `name` and `roll`.....");
	}
	public Student(String name, int rollNumber, String collegeName) {
		this.name = name;
		this.rollNumber = rollNumber;
		this.collegeName = collegeName;
		System.out.println("Inside Parameterized Constructor.....");
	}
}




`this` keyword:-
================
public class Student {
	String name;
	int rollNumber;
	String collegeName;
	
	public Student() {
		System.out.println("Inside no-argument constructor....");
	}
	void printThis() {
		System.out.println(this);
	}
}

public class Main {
	public static void main(String[] args) {
		Student st = new Student();
		System.out.println(st);//com.chitkara.Student@24d46ca6
		
		st.printThis();//com.chitkara.Student@24d46ca6
	}
}

Java Bean class:-
===============
Rules:-
------
- Class should be public.
- Members of class (instance variables) should be private.
- There should be a no-argument constructor.
- Argument constructor is optional.
- getter-setter method for instance variable.



Creation:-
---------
package com.masai;

public class Student {
	private String name;
	private int rollNumber;
	private String collegeName;
	
	public Student() {
	}

//	Optional (Argument Constructor)
	public Student(String name, int rollNumber, String collegeName) {
		this.name = name;
		this.rollNumber = rollNumber;
		this.collegeName = collegeName;
	}

	public String getName() {
		return name;
	}
	public void setName(String name) {
		this.name = name;
	}

	public int getRollNumber() {
		return rollNumber;
	}
	public void setRollNumber(int rollNumber) {
		this.rollNumber = rollNumber;
	}

	public String getCollegeName() {
		return collegeName;
	}
	public void setCollegeName(String collegeName) {
		this.collegeName = collegeName;
	}
	
	//Optional Method
	public String printDetails() {
		return "Student [name=" + name + 
				", rollNumber=" + rollNumber + 
				", collegeName=" + collegeName + "]";
	}
	
	
}

Create a calculator
======================
- Create a Calculator class:-
	- Implement four methods:
		* add(double a, double b)
		* subtract(double a, double b)
		* multiply(double a, double b)
		* divide(double a, double b)

- In main():
	Display a menu to the user:
	1. Add
	2. Subtract
	3. Multiply
	4. Divide
	5. Exit


- Take choice input from the user using Scanner.
- Take two numbers as input.
- Call the corresponding method based on the user's choice using a switch-case statement.
- Loop until the user chooses Exit.
- close the scanner class using scanner.close();
Solution:-
=========
package com.chitkara;

public class Calculator {
	double add(double x, double y) {
		return x + y;
	}
	double subtract(double x, double y) {
		return x - y;
	}
	double multiply(double x, double y) {
		return x * y;
	}
	double devide(double x, double y) {
		if (y == 0) {
			System.out.println("Can not devide by 0.");
			return 0;
		}
		return x / y;
	}
}

package com.chitkara;
import java.util.Scanner;
public class Main {
	
	public static void main(String[] args) {
		Scanner sc = new Scanner(System.in);
		Calculator c = new Calculator();
		int choice;
		do {
			System.out.println("Please provide your choice:-");
			System.out.println("=============================");
			System.out.println("1. Add");
			System.out.println("2. Subtract");
			System.out.println("3. Multiply");
			System.out.println("4. Devide");
			System.out.println("5. Exit");
			
			choice = sc.nextInt();
			if(choice >=1 && choice <=4) {
				System.out.println("Inter first Number:-");
				double x = sc.nextDouble();
				
				System.out.println("Inter second Number:-");
				double y = sc.nextDouble();
				System.out.println("***********************************************");
				switch (choice) {
				case 1->System.out.println("Result:- "+c.add(x,y));
				case 2->System.out.println("Result:- "+c.subtract(x,y));
				case 3->System.out.println("Result:- "+c.multiply(x,y));
				case 4->System.out.println("Result:- "+c.devide(x,y));
				}
				
				
			}else if(choice !=5){
				System.out.println("Invalid choice..");
			}
			
		} while (choice !=5);
		
		System.out.println("Thanks for using...");
		sc.close();
		
	}
}
